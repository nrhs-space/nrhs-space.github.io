病案管理

📒法律与时间相关

该类目以题库为基础,分类了与法律，时间规定相关的内容。

  1. 1982年卫生部颁布(《全国医院工作条例、医院工作制度与医院工作人员职责》),要求医院必须建立病案科。
  2. (2009)年12月26日颁发的《中华人民共和国侵权责任法》
    • 第61条规定,医疗机构必须按规定填写并妥善保管病历资料
    • 第54条规定,患者在诊疗活动中受到损害,医疗机构及其医务人员有过错的,由医疗机构承担赔偿责任
  3. (2012)年,原卫生部公布《医院评审标准》对病案管理人员做了要求。
  4. (2002)年,国务院颁布《医疗事故处理条例》，生效的时间是(2002年9月1日)
    • 第8条明确规定,医疗机构应当按照国务院卫生行政部门的规定要求,书写并妥善保管病历资料
    • 明确规定:“严禁涂改、伪造、隐匿、销毁或者抢夺病历资料”
  5. 作为CIS核心的电子病案早期尝试可追溯到(20世纪70年代)
  6. 20世纪60年代美`国麻省总医院开发完成的最早期电子病案系统之一是(COSTAR)
  7. 《中华人民共和国电子签名法》实施日期是(2005年4月1日)
  8. 1995年,我国卫生部提出了(金卫工程),该工程中将电子病案系统作为重点研究课题之一
  9. 2013年的《医疗机构病历管理规定》明确了对同一患者建立唯一的标识号码。1987即开始颁布
    • 已建立电子病历的医疗机构,应当将病历标识号码与患者(身份证号)编号相关联。
    • 明确指出“医疗机构应当建立病历管理制度,设置专门部门或者配备专(兼)职人员具体负责本机构病历和病案的保存与管理工作”
    • 住院病历由(所在病区)统一保管。
    • 《医疗机构病历管理规定》施行的时间是2002年9月1日
    • 要求借阅病历应当在(3)个工作日内归还
    • 纸质病历和电子病历具有(同等效力)
  10. 光盘存储技术目前应用广泛,资料显示光盘的使用寿命为10~20年
  11. 目前我国使用的全国统一病案首页是(2012)版病案首页
  12. 20世纪80年代中期,政府医疗卫生政策变化催生了临床信息系统的代表国家是(美国)
  13. 《三级综合医院评审标准实施细则(2011年版)》,其中的C级标准要求
    • 病案科非相关专业人员小于(50%)
    • 病案首页诊断填写完整,主要诊断的正确率达到(100%)

    • 住院病历的保存期不得少于30年

系列编号

  • 每次都给一个新号
  • 可减少工作流程，病案磨损等。但资料分散，不利于管理，查找…

单一编号

  • 将一个患者的所有医疗记录集中在一个病案号中管理。
  • 集中管理，保持了连续性与完整性。但工作流程繁琐，病案容易磨损…

系列单一编号

  • 将前两种编号结合起来。每次患者入院发一个新号，然后将旧号并入新号
  • 浪费人力物力，又降低了病案供应速度

P. 大多数医院均采用单一编号

一号集中制

全面搜集有关患者的医疗信息

  • 患者所有诊疗记录均保存在一个编号当中
  • 保证了资料的完整性，连续性

两号集中制

  • 住院记录与门诊记录分别编号，但病案却集中在一个编号中管理
  • 一旦发给住院号，则全使用住院号
  • 保证了资料的完整性，连续性

一号分开制

  • 住院病案与门诊病案分开管理，但使用同一个病案号
  • 保证了安全，但增加了工作量

两号分开制

  • 住院病案与门诊病案分别编码，单独存放，互不关联

📒指标与要求

该类目以题库与教材为基础，分类了一些大家喜闻乐见的~死规定~好要求

病案委员会

  • 由院长，临床科室等职能科室专家与病案科主任组成
  • 每年至少召开1~2次会议
  • 病案科是病案委员会的办事机构

三级医院

  • 科主任应具有本专业高技术职务任职资格者承担
  • 无菌手术切口感染率≤0.5%

二级医院

  • 二甲及以上医院专门从事住院病历管理的人员与医院病床比不得少于1:50
  • 病案科主任应具有本专业中技术职务任职资格者承担

其他

  1. 病案科应设有供医务人员讨论、分析、参阅病案的阅览室,其总面积为20~50m²
  2. 病案科工作人员需有独立的办公室,每人最少应拥有的工作空间面积为6m²
  3. 发达国家医院病床与病案管理人员的配备比例为10~15:1
  4. 环节质量控制的覆盖率应为35%~45%
  5. 终末质量监控的覆盖率应大于等于70%
  6. 随诊工作开展较好的医院,随诊率一般应≥95%
  7. 病案库房

  • 建筑耐火等级为一级
  • 与周围的建筑之间的防火间距应不小于30~35m
  • 温度标准为14℃~20℃,相对湿度为45%~ 60%。
  1. 病案科内至少应有贮存5年以上常用病案的空间
  2. 对于病案而言,不适宜的库房温度是指高于30℃，低于0℃
  3. 从事门诊病历管理的人员与医院日均门诊量的比不得少于1:300
  4. 病案科室主任应至少要有从事本专业工作5年以上

  6. 住院病案于患者出院24h内交送病案科
  7. 挂号员每日退号率为≤2%
  8. 医师因工作外出、出国或生病等合理原因长期停诊或短期停诊,必须提前3天向挂号处和医务部报批

医院统计

本章涉及较多统计学知识，可能我也有部分错误。注意查看。

一些统计学术语

  1. 计量资料：是对每个观察单位用定量方法测定某项指标量的大小所得到的资料，有单位，有确切值（身高体重）。
  2. 数值变量：说明事物数字特征，由数值变量的测量值构成的资料称为数值变量资料（定量资料）。大多数的数值变量为连续型变量,如身高、体重…
  3. 分类变量：说明事物类别的一个名称（疗效，行业，性别，血型）。也为计数资料。
  • 有序：取值各类别间存在一定程度差异（学历），它也是等级资料。
  1. 定量资料：以数字形式表现出来的研究资料
  • 离散型：在离散点上分布的原始数据(家庭人口)
  1. 定性资料：以文字，图形，录像，音频等非数字形式表现的资料
  2. 等级资料：具有一定级别的资料，没有确切的值（疗效）
  3. 计数资料：将观察单位按照其性质与类别分组，然后分析各组情况。只在乎数量，不在乎具体指标性质
  4. 定量变量：即连续量，由测量或计数，统计得到的量（血脂），它具有数值特征。
  5. 离散型资料：其取值可以一一列举出来
  6. 连续型资料：取值为范围内任意值
  7. 二项分类资料：将所有文本划分为两个互不相容的类别的方法。

概念及含义

  1. 归纳法：平均分析，相关分析，相对分析，回归分析
  2. 医院统计分析的基本内容：
  • 分析事物内部构成
  • 分析事物的相互关系
  • 分析计划指标完成情况
  • 综合对比分析
  1. 统计量是描述样本特征的指标
  2. 有代表性的样本指的是依照随机原则抽取总体中的个体
  3. 统计资料的类型包括数值变量资料和分类变量资料
  4. 老龄化是指65岁及以上老龄人口占总人口的比重达到7%
  5. 病死率:疾病的预后
  6. 死亡率:人群中因该病死亡的频数

法律与规定

  1. 《国家卫生统计调查制度》制定的依据是《全国卫生统计工作管理办法》
  2. 医院统计工作的主要内容是在《全国卫生资源与医疗服务调查制度》指导下开展的

其他部分

  1. 统计工作的步骤中,最关键的一步是设计
  2. 统计分析的主要内容是统计描述与统计推断
  3. 等级分组资料兼有计量资料和计数资料的一些性质
  4. 相对数指标分析中标化率不反映某现象发生的实际水平
  5. 构成比通常以100%为比例基数
  6. 计算相对数的目的是便于比较
  7. 计算标化率的目的是消除资料内部构成不同的影响,使率具有可比性
  8. 描述分类变量资料的主要统计指标是相对数
